# 数值稳定性 & 初始化

------

# 🔹 1. 核心问题：数值不稳定

## ❗ 深层网络的本质

$$
h^{(L)} = f_L \circ f_{L-1} \circ \cdots \circ f_1(x)
$$

👉 本质是：

```
多层连续乘法（矩阵乘法 + 导数链式法则）
```

------

## 🔸 梯度形式（关键🔥）

$$
\frac{\partial L}{\partial W^{(l)}} = M^{(L)} M^{(L-1)} \cdots M^{(l+1)} v^{(l)}
$$

👉 也就是：

```
很多矩阵连乘
```

------

## 🔸 导致的问题

### 1️⃣ 梯度消失（vanishing）

```
连续乘 <1 → 指数级变小 → 学不动
```

👉 常见原因：

- sigmoid/tanh 饱和
- 权重太小

------

### 2️⃣ 梯度爆炸（exploding）

```
连续乘 >1 → 指数级变大 → 参数崩掉
```

👉 常见原因：

- 权重太大
- 深层网络

------

📌 本质总结：

> **深度网络 = 连续乘法 → 极易数值失控** 

------

# 🔹 2. 核心目标（非常重要🔥）

> ❗让每一层的“输出”和“梯度”都保持稳定

换句话说：

```
不变大（防爆炸）
不变小（防消失）
```

------

# 🔹 3. 初始化方法（第一类解决方案）

------

## 🔸 Xavier 初始化

$$
\mathrm{Var}(W) = \frac{2}{n_{\text{in}} + n_{\text{out}}}
$$

👉 作用：

- 前向：输出方差稳定
- 反向：梯度方差稳定

------

👉 本质：

```
每一层 ≈ 不放大、不缩小
```

------

## 🔸 He 初始化（更常用🔥）

$$
\mathrm{Var}(W) = \frac{2}{n_{\text{in}}}
$$

👉 专为 ReLU 设计

------

## 🔸 本质总结

> **初始化 = 控制“起点尺度”**

------

# 🔹 4. 激活函数（第二类）

------

## ❌ sigmoid（容易出问题）

- 大/小输入 → 梯度≈0
- 多层后 → 梯度消失 

------

## ✅ ReLU（现代默认）

- 不饱和（正区间）
- 梯度稳定

------

## 🔥 结论

```
激活函数会影响梯度传播！
```

------

# 🔹 5. 归一化方法

------

## 🔸 Batch Normalization（BN）

$$
x′=x−μσ
$$

👉 作用：

- 控制每层分布
- 防止数值漂移
- 加速训练

------

## 🔸 Layer Normalization（LN）

👉 Transformer 用这个

------

## 🔥 本质

> **让每一层“重新归一化”，防止数值越传越偏**

------

# 🔹 6. 改结构：把乘法变加法（重点🔥）

------

## 🔸 Residual（残差连接）

$$
y=x+F(x)
$$



------

### 为什么有效？

👉 原本：

```
梯度 = 连乘（容易消失/爆炸）
```

👉 加残差：

```
梯度 = 乘法路径 + “直接通路”
```

------

👉 即使：

- F(x) 很小
- 梯度很弱

👉 仍然可以：

```
通过 x 直接传递
```

------

## 🔥 本质

> ❗**用“加法路径”缓解“连乘问题”**

------

# 🔹 7. 其他稳定技巧（工程常用）

------

## 🔸 Gradient Clipping（梯度裁剪）

```
clip_grad_norm_(params, max_norm)
```

👉 防止：

```
梯度爆炸
```

------

## 🔸 对数变换（log trick）

$$
log⁡(ab)=log⁡a+log⁡b
$$

👉 用于：

- softmax
- 概率计算

👉 作用：

```
乘法 → 加法 → 防溢出
```

------

## 🔸 RNN/LSTM 结构改进

👉 用门控机制：

```
控制信息流动（避免梯度消失）
```

------

# 🔹 8. 总体方法总结（最重要🔥）

| 类别     | 方法        | 解决问题     |
| -------- | ----------- | ------------ |
| 初始化   | Xavier / He | 控制初始尺度 |
| 激活函数 | ReLU        | 防梯度消失   |
| 归一化   | BN / LN     | 稳定分布     |
| 结构     | Residual    | 打破连乘     |
| 技巧     | Clipping    | 防爆炸       |
| 数学     | log变换     | 防数值溢出   |

------

# 🔥 9. 一句话终极总结

> **深度学习的不稳定，本质来自“连乘”，解决方法就是：控制尺度（初始化/归一化）、改变结构（残差）、或改变计算形式（log/加法）。**

------

# 🔥 10. 现代大模型的“三件套”

👉 Transformer 为什么能训练很深：

```
LayerNorm + Residual + 合理初始化
```

------

# 🔥 最后给你一个理解框架

```
初始化：起点不炸  
激活：中间不灭  
归一化：每层不漂  
残差：梯度不断  
```
